This Is A Demo Page

Try navigating this page with a screen reader, or with an extension that fuzzes the page to mimic a low vision impairment. What are the issues you run into? Were you able to navigate to all the elements on the page? How can these issues be mitigated?

This section has a chart

Note that this workshop doesn't address the possible accessibility issues within the chart itself. This example uses Chart.js and it has some documentation on accessibility that can be found here.

What about the colors?

What do you think about the colors of that chart? How might you correct that to make it more accessible?
